# C-1semester
Лабораторные работы на C (1 семестр).

lab1 - Значение функции.

lab2 - Тип треугольника по сторонам.

lab3 - Пройденный путь с разными скоростями.

lab4 - Решение квадратного уравнения.

lab5 - Тип студента по оценкам.

lab6 - Объем шкафа.

lab7 - Положение квадрата и окружности.

lab8 - Пройдет ли контейнер в дверь?

lab9 - Простейший калькулятор.

lab_10 - Алгоритм Евклида.

lab_11 - Алгоритм поиска Чисел Фибоначчи.

lab_12 - Сумма цифр числа.

lab_13 - Пользователь вводит число N (0 < N < 27) реализовать программу из диапозона 0-999, сумма цифр которых равна N.

lab_14 - Вычислить значение функции y от x. Значение переменной x и параметров a и b вводить с клавиатуры (Параметр a вещественного типа , параметр b целочисленный).

lab_15 - Область значений функции.

lab_16 - Алгоритм поиска min и max элементов в массиве и их позицию.

lab_17 - По заданным дввум массивам целых чисел сформировать массив-обьединение (без повторений) и массив-пересечение.

lab_18 - Дано целое число a (0 < a < 10^9), среди цифр которого нет цмфры 0. Составить число b из 9 цифр, в каждой позиции которого значение равно количеству таких цифр в исходном числе.

lab_19 - В массиве каждый третий элемент заменяет полусуммой двух предыдущех, а стоящий перед ним - полусуммой соседних с ним эелементов.

lab_20 - Реализовать бинарный поиск. Найти элемент в упорядоченном массиве.
